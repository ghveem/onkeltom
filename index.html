NDLA image-api

Bildebatch-opplaster

Statisk GitHub Pages-app for opplasting av mange bilder til NDLA sitt image-api. Tittel blir hentet fra filnavnet, og serienavn legges til som nøkkelord på hvert bilde.

Denne versjonen er laget for ren GitHub Pages-deploy. Ingen backend er nødvendig, men du må fylle inn Auth0-konfig i config.js.
